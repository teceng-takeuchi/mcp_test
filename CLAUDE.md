# CLAUDE.md

このファイルは、このリポジトリのコードを扱う際のClaude Code (claude.ai/code) への指針を提供します。

## プロジェクト概要

これは「mri」という名前の最小限のnpmプロジェクトで、初期段階にあるようです。現在、リポジトリには依存関係のないpackage-lock.jsonファイルのみが含まれています。

## MCPについて

このプロジェクトにおいてMCPという単語はMaritime Connectivity Platformの事を指します。

## 開発セットアップ

プロジェクトにpackage.jsonファイルが不足しているため、このコードベースで作業する際の最初のステップは以下の通りです：

1. タスクに必要な場合はpackage.jsonファイルを作成する
2. ユーザーの要件に基づいてプロジェクトの目的と技術スタックを決定する
3. 必要に応じて適切な設定ファイルをセットアップする

リポジトリは現在最小限の構造を持っています：

## タスク完了時の通知

タスクが完了したら、以下のコマンドを実行して通知音を鳴らしてください：
```bash
npm run notify
```

注意：このコマンドは手動で実行する必要があります。タスクやコマンドの実行後に自動的に音を鳴らしたい場合は、以下の方法を使用してください：

### 自動通知の使い方

1. **シェルスクリプトを使う方法**：
   ```bash
   ./run-with-notification.sh <実行したいコマンド>
   # 例: ./run-with-notification.sh npm test
   ```

2. **npmスクリプトを使う方法**：
   ```bash
   npm run test:with-notify
   npm run build:with-notify
   ```

3. **コマンドチェーンを使う方法**：
   ```bash
   <実行したいコマンド> && npm run notify
   # 例: npm test && npm run notify
   ```
